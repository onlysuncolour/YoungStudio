## 题目描述

给你一个整数数组 nums ，数组中的元素 互不相同 。返回该数组所有可能的子集（幂集）。  
  
解集 不能 包含重复的子集。你可以按 任意顺序 返回解集。  
  
   
  
示例 1：  
  
输入：nums = [1,2,3]  
输出：[[],[1],[2],[1,2],[3],[1,3],[2,3],[1,2,3]]  
示例 2：  
  
输入：nums = [0]  
输出：[[],[0]]  
   
  
提示：  
  
* 1 \<= nums.length \<= 10  
* -10 \<= nums[i] \<= 10  
* nums 中的所有元素 互不相同  

## 本人题解

```javascript
/**
 * @param {number[]} nums
 * @return {number[][]}
 */
var subsets = function(nums) {
    var subset = [];
    for (var i = 0; i < nums.length; i++) {
      subset = [...subset, ...getsubs(nums.slice(i, nums.length))]
    }
    subset.push([]);
    return subset
};

var getsubs = function(nums) {
  var subs = [];
  if (nums.length === 1) {
    return [nums]
  } else {
    for (var j = 1; j < nums.length; j++) {
      subs = [...subs, ...getsubs(nums.slice(j, nums.length))];
    }
    for (var i = 0; i < subs.length; i++) {
      subs[i].push(nums[0])
    }
    subs.push([nums[0]])
  }
  return subs
}
```
