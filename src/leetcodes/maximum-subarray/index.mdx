## 题目描述

给你一个整数数组 nums ，请你找出一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。  
  
子数组是数组中的一个连续部分。  
  
   
  
示例 1：  
  
输入：nums = [-2,1,-3,4,-1,2,1,-5,4]  
输出：6  
解释：连续子数组 [4,-1,2,1] 的和最大，为 6 。  
示例 2：  
  
输入：nums = [1]  
输出：1  
示例 3：  
  
输入：nums = [5,4,-1,7,8]  
输出：23  
   
  
提示：  
  
* 1 \<= nums.length \<= 10<sup>5</sup>  
* -10<sup>4</sup> \<= nums[i] \<= 10<sup>4</sup>  

## 本人题解

```javascript
/**
 * @param {number[]} nums
 * @return {number}
 */
var maxSubArray = function(nums) {
  let max = 0, minus = 0, maxs = [];
  nums.forEach(n => {
    if (n > 0) {
      if (max + minus + n < max) {
        maxs.push(max)
        if (max + minus + n < n) {
          max = n
        } else max = max +minus + n
      } else {
        if (max + minus + n < n) {
          max = n;
        } else {
          max = max + minus + n
        }
      }
      minus = 0
    } else {
      if (max > 0) {
        minus += n
      }
    }
  })
  if (max > 0) {
    maxs.push(max)
  }
  if (maxs.length == 0) {
    maxs = nums
  }
  // console.log(maxs)
  return maxs.sort((a,b) => a-b).pop()
};
```
