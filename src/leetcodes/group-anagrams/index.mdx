## 题目描述

给你一个字符串数组，请你将 字母异位词 组合在一起。可以按任意顺序返回结果列表。  
  
字母异位词 是由重新排列源单词的所有字母得到的一个新单词。  
  
   
  
示例 1:  
  
输入: strs = ["eat", "tea", "tan", "ate", "nat", "bat"]  
输出: [["bat"],["nat","tan"],["ate","eat","tea"]]  
示例 2:  
  
输入: strs = [""]  
输出: [[""]]  
示例 3:  
  
输入: strs = ["a"]  
输出: [["a"]]  
   
  
提示：  
  
* 1 \<= strs.length \<= 10<sup>4</sup>  
* 0 \<= strs[i].length \<= 100  
* strs[i] 仅包含小写字母  

## 本人题解

```javascript
/**
 * @param {string[]} strs
 * @return {string[][]}
 */
var groupAnagrams = function(strs) {
  let result = [], help = [];
  strs.forEach(s => {
    let ts = s.split('').sort().join('')
    let i = help.indexOf(ts)
    if (i == -1) {
      result.push([s])
      help.push(ts)
    } else {
      result[i].push(s)
    }
  })
  return result
};
```
