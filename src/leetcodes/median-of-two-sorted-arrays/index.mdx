## 题目描述

给定两个大小分别为 m 和 n 的正序（从小到大）数组 nums1 和 nums2。请你找出并返回这两个正序数组的 中位数 。  
  
算法的时间复杂度应该为 O(log (m+n)) 。  
  
   
  
示例 1：  
  
输入：nums1 = [1,3], nums2 = [2]  
输出：2.00000  
解释：合并数组 = [1,2,3] ，中位数 2  
示例 2：  
  
输入：nums1 = [1,2], nums2 = [3,4]  
输出：2.50000  
解释：合并数组 = [1,2,3,4] ，中位数 (2 + 3) / 2 = 2.5  
  
   
  
提示：  
  
* nums1.length == m  
* nums2.length == n  
* 0 \<= m \<= 1000  
* 0 \<= n \<= 1000  
* 1 \<= m + n \<= 2000  
* -10<sup>6</sup> \<= nums1[i], nums2[i] \<= 10<sup>6</sup>  


## 本人题解

```javascript
function findMedianSortedArrays(nums1, nums2) {
    let result, cur = 0;
    nums1.forEach(n => {
        while (nums2[cur] < n && nums2[cur] != undefined){
            cur++
        }
        nums2.splice(cur, 0, n)
    })
    if (nums2.length % 2 == 0) {
        result = (nums2[nums2.length/2] + nums2[nums2.length/2-1]) / 2
    } else 
        result = nums2[Math.floor(nums2.length/2)]
    return result
};
```
